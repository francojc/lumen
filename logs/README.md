# Logs

This directory tracks lightweight operational status notes for project continuity.

## Cadence

- **Weekly review:** one file per week in `logs/weekly/`
- **Session notes:** one file per focused work block in `logs/sessions/`

## File Naming

- Weekly: `YYYY-Www.md` (example: `2026-W16.md`)
- Session: `YYYY-MM-DD-topic.md` (example: `2026-04-17-v03-planning.md`)

## Minimum Content

### Weekly review

- What changed
- What is blocked or at risk
- Priorities for next week

### Session log

- Objective
- Actions taken
- Decisions made
- Next actions

Use templates from `logs/templates/`.
